# MongoDB-Social-Network

## Description

A application using MongoDB to create a social media
backend so users can sign up, share their thoughts and react to other users thoughts. This application was built to help further my knowledge with no structure databases.

## Installation

Run "npm i" on the package.json to have access to this application. You can use Insomina to test the user routes due to a lack of front -end.
